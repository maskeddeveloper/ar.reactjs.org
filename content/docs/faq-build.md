### كيف يمكنني كتابة التعليقات في JSX؟ {#how-can-i-write-comments-in-jsx}

```jsx
<div>
  {/* التعليقات تأتي هنا */}
  Hello, {name}!
</div>
```

```jsx
<div>
  {/* تعمل أيضًا 
  لأجل التعليقات متعددة الأسطر */}
  Hello, {name}! 
</div>
```
